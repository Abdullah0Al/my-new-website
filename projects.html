Expense Tracker App

February 2025

The Expense Tracker App is a full‐stack web application built with Python (Flask), SQLite, Bootstrap, and Chart.js. It helps users manage personal expenses, featuring secure user authentication, comprehensive expense management (add, edit, delete), date‐range filtering, and data visualization with interactive charts. A CSV export option allows for further data analysis.

  • Key Features: Secure login, expense filtering, interactive dashboard, CSV export.
  • Technologies: Flask, SQLite, Flask‐Login, Chart.js, Bootstrap 5.

Murabi

February 2025

Murabi is a web-based livestock management solution that streamlines health monitoring, breeding, and financial records on a single platform. Our goal is to modernize farm operations with real-time data and easy reporting tools. Currently in progress, it supports Arabic UI and adheres to Saudi regulations.
