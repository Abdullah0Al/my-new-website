Digital Image Processing and Analysis

Fall 2024 (First Semester 1446/1447)

This project focuses on applying image processing techniques to a number plate image. Key steps included grayscale conversion, noise reduction with Gaussian filtering, high-pass filtering to enhance edges, adaptive thresholding for binarization, morphological operations to remove noise, and connected component analysis to count characters/digits.

Highlights:

  • Implemented high-pass filtering and adaptive thresholding.
  • Used morphological operations to remove noise.
  • Segmented sub-images for Arabic/English letters and digits.

AI-Powered Real Estate Agent (Information Retrieval)

Fall 2024

This project leverages Information Retrieval (IR) and Natural Language Processing (NLP) techniques to build an AI-powered real estate agent. It processes user queries (e.g., "2-bedroom apartment in Jeddah under 7,000 SAR") and provides relevant property recommendations by analyzing TF-IDF vectors, applying filters (budget, location, rooms), and ranking results via cosine similarity.

Highlights:

  • Data preprocessing (tokenization, stop word removal, lemmatization).
  • TF-IDF vectorization and cosine similarity for ranking.
  • Basic user interaction flow for refining searches.
